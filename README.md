# LinkHub

A playful link-in-bio landing built with React + TanStack Start, Vite, and Tailwind CSS. Includes hero sections, features, testimonials, and a footer—ready to wire up real routes and animations.

## Features
- Colorful gradient background with glassy UI accents
- Responsive navigation with logo + brand text (LinkHub) and menu links (Pricing, Login, Sign Up)
- Hero 1: headline, subheading, and a URL input with a fixed `linkhub.link/` prefix + “Claim your URL”
- Hero 2: swapped layout with a “Start for free” CTA
- Features: three-item grid with Rive animation placeholders
- Testimonials: single large Rive animation placeholder
- Footer with simple links
- TanStack Start entry files for SSR-ready routing

## Tech stack
- React 19, TypeScript 5
- TanStack React Start + Router
- Vite 7
- Tailwind CSS 4
- pnpm

## Getting started

Prerequisites: pnpm

```fish
# Install deps
pnpm install

# Copy and configure wrangler.jsonc
cp wrangler.example.jsonc wrangler.jsonc
# Edit wrangler.jsonc and replace REPLACE_WITH_YOUR_DATABASE_ID with your actual Cloudflare D1 database ID

# Start dev server (http://localhost:3000)
pnpm dev

# Build for production
pnpm build
```

Notes:
- The dev server port is configured in `vite.config.ts` (default: 3000).
- A preview script isn’t added by default; you can run `vite preview` manually if needed.

## Routing
- File-based routes via TanStack Router in `src/routes/`.
- `__root.tsx` sets up the HTML shell and scripts; `index.tsx` renders the landing page.
- `routeTree.gen.ts` is auto-generated—safe to ignore in Git if you prefer.

## Styling
- Tailwind CSS v4 is imported globally in `src/styles.css`.
- The landing uses utility classes for layout and the gradient background.

## Logo
- Use `<Logo className="h-8 w-8" />` from `src/lib/Logo.tsx`, which renders `src/assets/logo.svg`.

## URL input prefix
- The input shows a non-editable prefix `linkhub.link/` and a controlled text value after it.
- Padding uses a character-based width (`pl-[14ch]`) to keep spacing aligned with the prefix. Adjust as needed.
- To construct the final URL, combine on submit: `https://linkhub.link/${handle}`.

## Rive placeholders
- Sections include clear placeholders for future Rive animations.
- When ready, add `@rive-app/react-canvas` and mount your animations inside the placeholder containers.

## Deploy
- Works great on Vercel/Netlify/Cloudflare Pages. For basic static output use `pnpm build`.
- If you plan to use SSR, follow TanStack Start’s adapter guidance for your platform.
